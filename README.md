## 🏗️ Project Architecture

This project integrates raw sales data with modern analytics tools to generate actionable insights for BMW’s leadership and dealers:

* 🔄 **Data Extraction**: Import BMW global sales CSV
* 🧹 **Data Cleaning (Python/Excel)**: Handle missing records, standardize columns, and preprocess sales data
* 🗃️ **Data Modeling (Power BI)**: Create relationships between sales, regions, models, and dealer networks
* 📊 **Visualization (Power BI)**: Build dashboards for trends, KPIs, and comparisons

---

## ✨ Key Features

* 🚗 **Model-wise Sales Breakdown**: Track revenue, units sold, and growth for each BMW model
* 🌍 **Geographic Insights**: Regional and country-level performance mapping
* 🏬 **Dealer Analysis**: Top-performing dealers and market penetration
* 📈 **Time-Series Trends**: Monthly/quarterly sales trajectory with YoY comparisons
* 🏷️ **Profitability & Revenue Insights**: Contribution of premium vs. standard models
* ⚡ **Interactive Filters**: Dynamic drilldowns by year, region, or vehicle model

## 📂 Dataset

The BMW sales dataset includes:

* **Sales Figures**: Units sold, revenue, profit margins
* **Vehicle Models**: X-series, 3-series, i-series, etc.
* **Regions**: Europe, North America, Asia, Middle East, etc.
* **Dealership Data**: Dealer ID, location, sales performance
* **Timeline**: Monthly/quarterly/yearly trends
